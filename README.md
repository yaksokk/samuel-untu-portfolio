This is my personal portfolio website — built to introduce myself, share what I've learned, and showcase my journey as a developer.

## Built With

- **React 19** + **Vite 7**
- **Tailwind CSS v4**
- **pnpm**
- **react-icons v5**

## Getting Started

```bash
pnpm install
pnpm dev
```
